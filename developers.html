Authentication

All API requests are authenticated using Bearer tokens. Generate your API key from Dashboard ‚Üí Settings ‚Üí API Keys.

  • ‚úì Rate limit: 1,000 requests/min (Business), 10,000 (Commerce)
  • ‚úì All responses in JSON format
  • ‚úì TLS 1.3 required for all requests
  • ‚úì Sandbox environment: api-sandbox.nexlance.com
// Install: npm install @nexlance/sdk import { NexlanceClient } from '@nexlance/sdk'; const client = new NexlanceClient({ apiKey: 'nlk_live_your_api_key_here' }); // List all sites const { sites } = await client.sites.list(); console.log(sites); // Create a new page const page = await client.pages.create({ siteId: 'site_abc123', title: 'About Us', slug: 'about' });
